## Run Creation Overview

Creating a Psyche training run involves:

1. Setting up join authorization (permissioned or permissionless)
2. Creating the run on-chain with a unique run ID
3. Configuring the run with your model and training parameters
4. Unpausing the run to allow clients to join
5. (Optional) Setting up token rewards and funding the treasury

## Responsibilities as a Run Owner

As the run owner, you'll be responsible for:

- Maintaining the run configuration and adjusting parameters as needed
- Managing client authorization (for permissioned runs)
- Monitoring run health and participant activity
- Funding the treasury if distributing token rewards
- Pausing/resuming the run when necessary
